# Sources for bulimov.ru site

Uses [Hugo static site generator](gohugo.io)


## Setting things up

* Install hugo 0.80+
* Install *ghp-import* using `pip3 install ghp-import`


## Adding new content

### Post

`hugo new post/some-slug.md`


## Publishing

```
make github
```
